# PDF Master - Play Store Assets Guide

## App Icon
The app icon has been created as an adaptive icon with:
- **Foreground**: White document with blue "PDF" text
- **Background**: Solid blue (#2563EB)
- **Monochrome**: For Android 13+ themed icons

Files created:
- `app/src/main/res/drawable/ic_launcher_foreground.xml`
- `app/src/main/res/drawable/ic_launcher_background.xml`
- `app/src/main/res/drawable/ic_launcher_monochrome.xml`
- `app/src/main/res/mipmap-anydpi-v26/ic_launcher.xml`

## Play Store Listing Assets Required

### 1. App Icon (512x512)
Export the adaptive icon as a 512x512 PNG for Play Store.

### 2. Feature Graphic (1024x500)
**Design Specs:**
- Size: 1024 x 500 pixels
- Background: Gradient from #2563EB to #7C3AED
- Content:
  - App name "PDF Master" in bold white
  - Tagline: "All-in-One PDF Solution"
  - Show 3 phone mockups with app screens
  - Icons representing: View, Edit, Scan, Merge

### 3. Screenshots (Min 4, Max 8)
**Required sizes:**
- Phone: 1080 x 1920 (or 9:16 ratio)
- Tablet 7": 1200 x 1920
- Tablet 10": 1600 x 2560

**Recommended Screenshots:**
1. Home screen with file list
2. PDF Viewer with reading modes
3. Document Scanner in action
4. PDF Editor with text tools
5. Merge/Split tools
6. Annotation tools
7. Signature feature
8. Settings with premium options

### 4. Short Description (80 chars max)
```
PDF viewer, editor, scanner & converter. Merge, split, compress PDFs easily.
```
